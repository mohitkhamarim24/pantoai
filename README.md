# Panto AI Repo Reviewer

## Features
- GitHub & GitLab OAuth Login
- Fetch user's repositories
- Enable/disable "Auto Review"
- Profile overview
- Docker-ready
- Hardcoded fallback for repo details (when API fails)

## Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- OAuth (GitHub/GitLab)
- Axios, React Router
- Docker & Docker Compose
